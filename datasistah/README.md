# Data Sistah

**AI Development & Consulting**

A professional website for Data Sistah, an AI development and consulting company specializing in building, improving, and supporting AI-powered software and data science applications.

## About

Data Sistah provides artificial intelligence (AI) development and consulting services centered on how AI is designed, developed, and applied within software-based products. Led by Tiffany Teasley, a working data scientist and AI developer with hands-on experience applying AI and data science to real product work.

## Services

- **AI Development**: AI feature development, data science solutions, enhancements to existing AI implementations
- **LLM Integration**: Working with large language models as part of AI-powered applications
- **AI Consulting**: Advising on AI and data science application design, reviewing implementations, guiding technical decisions

## Website Structure

- `index.html` - Main website file with all sections
